# Tema Software Engineer - Design System

## Overview

Tema telah diperbarui untuk mencerminkan estetika **Software Engineer** yang profesional, modern, dan teknis.

## Perubahan Tema

### 1. ✅ Warna (Color Palette)

**Primary Color - Tech Blue**

- `#3b82f6` - Modern tech blue (menggantikan cyan)
- `#2563eb` - Primary action color
- Lebih profesional dan tech-oriented

**Accent Color - Terminal Green**

- `#22c55e` - Terminal green untuk accent
- Memberikan sentuhan developer aesthetic

**Text Colors**

- Primary: `#1e293b` - Lebih gelap, lebih profesional
- Secondary: `#475569` - Lebih soft
- Tertiary: `#64748b` - Lebih light

### 2. ✅ Typography

**Font Families**

- **Sans**: Inter (modern, clean, professional)
- **Mono**: JetBrains Mono / Fira Code (developer fonts)
- **Display**: Inter untuk headings

**Improvements**

- Letter spacing: `-0.01em` untuk modern look
- Line height: `1.6` untuk readability
- Font weights: 300-700 untuk hierarchy

### 3. ✅ Code Elements

**Code Styling**

- Background: `#f1f5f9` untuk inline code
- Color: `#3b82f6` untuk code text
- Monospace font: JetBrains Mono

**CodeBlock Component (BARU)**

- Terminal-like appearance
- Window controls (red, yellow, green dots)
- Language indicator
- Dark theme (`#1e293b`)

### 4. ✅ Components Styling

**Buttons**

- Primary: Tech blue dengan shadow
- Hover effects dengan transform
- Font weight: 500 untuk clarity

**Cards**

- Subtle shadows
- Hover effects dengan elevation
- Clean borders

**Badges & Tags**

- Tech blue background
- Font weight: 600
- Letter spacing untuk clarity

**Timeline (Experience)**

- Tech blue dots dengan glow effect
- Modern arrow indicators

**Stats**

- Gradient text effect untuk values
- Professional appearance

### 5. ✅ Hero Section

**Background**

- Gradient dengan tech blue tones
- Radial gradients untuk depth
- Subtle patterns

### 6. ✅ Navigation

**Header**

- Active state dengan tech blue
- Underline indicator dengan rounded corners
- Font weight: 600 untuk active

## Design Principles

### 1. Professional & Clean

- Minimal design
- Clear hierarchy
- Consistent spacing

### 2. Technical Aesthetic

- Monospace fonts untuk code
- Terminal-like elements
- Tech color palette

### 3. Modern & Contemporary

- Inter font family
- Subtle gradients
- Smooth transitions

### 4. Developer-Friendly

- Code blocks dengan syntax highlighting ready
- Terminal aesthetics
- Tech-oriented colors

## Color Usage

### Primary Actions

- Buttons: `#3b82f6`
- Links: `#3b82f6`
- Active states: `#3b82f6`

### Accents

- Success/Positive: Terminal green

## Typography Scale

- **H1**: 3rem (48px) - Bold
- **H2**: 2.25rem (36px) - Bold
- **H3**: 1.875rem (30px) - Semibold
- **H4**: 1.5rem (24px) - Semibold
- **Body**: 1rem (16px) - Regular
- **Small**: 0.875rem (14px) - Regular
- **Code**: 0.875rem (14px) - Medium
